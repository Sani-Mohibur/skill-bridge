1. Home Page (Public)
Hero Section: A minimalist headline showcasing the platform's value, a central search input to lookup tutors immediately, and primary CTA buttons ("Find a Tutor" / "Become a Tutor").
Featured Categories: A clean grid layout highlighting top subject categories (e.g., Computer Science, Mathematics) fetched directly from the backend.
How It Works: A simple, high-contrast 3-step timeline/walkthrough showing how the process works for both students and tutors.
Success Stories: A clean layout of minimal card reviews showcasing positive feedback from students.


2. Tutor Search / Discovery Page (Public)
Filter Sidebar: Interactive controls to filter tutors by subject, availability, price range, and ratings.
Tutor Grid List: A responsive list of cards displaying tutor profiles, hourly rates, bio snippets, and a "View Profile" button.
Pagination / Infinite Scroll Control: Minimal layout to navigate between pages of search results.


3. Tutor Public Profile Page (Public)
Tutor Header Details: Main profile banner showing avatar, name, badges, bio, and overall rating.
Booking / Availability Calendar: A grid component displaying the tutor’s available time slots where students can click to book.
About & Qualifications Section: In-depth text detailing the tutor's background, experience, and subjects taught.


4. Login & Register Pages (Auth Gateway)
Credentials Form Card: Clean, centralized input card for email and password with real-time Zod validations.
Role Selector (Registration Only): Clear toggle buttons to let users choose whether they are joining as a "Student" or a "Tutor".


5. Student Dashboard (Protected)
Overview Stats Grid: Summary cards showing total hours learned, active bookings, and total spent.
My Bookings List: A clean table layout tracking active, pending, and past session enrollments with a "Cancel Booking" action.
Profile Settings Form: Isolated section to update personal details, change password, and manage avatars.


6. Tutor Dashboard (Protected)
Overview Performance Cards: High-level metrics tracking total earnings, total slots filled, and active students.
Manage Slots Grid: A calendar interface where tutors can create, open, or delete upcoming availability time slots.
Roster & Slot Attendees List: Deep-dive view showing exactly which students have booked specific slots, complete with the "Mark Session Completed" action.
